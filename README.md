# SocialMediaApp
Application of small social media. App downloads data from public API.
	
## Technologies
Project is created in:
Java 11, 
Spring 2.3.4,
Thymeleaf 3.0.9, 
PostgreSQL 12, 
Hibernate 6.0.13 Final, 
Junit 4.13, 
Hamcrest 1.3,
Mockito 1.10.19, 
Maven 3.81, 
IntelliJ IDEA, 
Git

	
## Setup
User should provide its own datasource, username and password for Postgresql db
